# Student Allowance Smart Contract

This smart contract is designed to manage student allowances, ensuring they can only spend within their allocated amounts. The contract demonstrates the use of `require()`, `assert()`, and `revert()` statements in Solidity.

## Contract Features

- **Set Allowance**: Assign an allowance to a student.
- **Spend Allowance**: Allow students to spend their allowance, ensuring they have sufficient funds.
- **Check Allowance**: Verify if a student has enough allowance to spend a specified amount.

## Deployment

You can deploy this contract using an Ethereum-compatible development environment. Below are the steps to deploy using [Remix IDE](https://remix.ethereum.org/):

1. Open [Remix IDE](https://remix.ethereum.org/).
2. Create a new file and paste the smart contract code.
3. Compile the contract using the Solidity compiler.
4. Deploy the contract and test the funstions created.
